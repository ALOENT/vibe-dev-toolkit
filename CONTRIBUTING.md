---

## ◆ How to Contribute

```bash
# 1. Fork the repo
# 2. Create your branch
git checkout -b add/your-contribution-name

# 3. Add your files or edit existing ones

# 4. Commit with a clear message
git commit -m "add: your-contribution-name"

# 5. Open a Pull Request
```

> Keep your PR focused — one contribution per PR makes reviews fast and clean.

---

## ◆ Commit Message Format

```
add:    new prompt or resource
fix:    typo or broken link
update: improved existing content
remove: outdated or irrelevant content
```

---

## ◆ Rules

- No paywalled or stolen content
- If it's someone else's resource — **credit them clearly**
- No spam, no self-promo without real value
- Keep it clean, keep it useful
